# -SISTEMA-DE-AUTENTICA-O-EM-DUAS-ETAPAS-COM-AN-LISE-DE-VOZ
Sistema de autenticação biométrica por frequência de voz. O usuário cadastra uma senha fixa e fala três palavras aleatórias exibidas na tela para liberar o acesso. A análise é feita localmente via Web Audio API e Whisper.cpp, com um agente de IA para calibração de voz. Stack: React, Node.js, PostgreSQL, Whisper.cpp e Ollama.
