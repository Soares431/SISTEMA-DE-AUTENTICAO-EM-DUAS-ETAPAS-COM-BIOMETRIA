# -SISTEMA-DE-AUTENTICA-O-EM-DUAS-ETAPAS-COM-BIOMETRIA
Arquitetura Inicial do Projeto

Projeto desenvolvido por Gustavo utilizando Vibe Coding com Claude.

A aplicação foi construída com Node.js e utiliza SQLite como banco de dados para testes. O sistema roda em ambiente local (localhost) para fins de desenvolvimento e verificação.

Funcionalidades
Cadastro de usuários
Sistema de verificação com senha gerada automaticamente
Identificação por matrícula
Integração com leitura de impressão digital (em desenvolvimento)
