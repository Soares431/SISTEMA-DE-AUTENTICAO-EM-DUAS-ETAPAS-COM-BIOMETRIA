1. Introdução

Este manual mostra como usar o sistema de controle de acesso do 5º CTA. Em cada seção você vai ver uma reprodução da tela do sistema com números vermelhos marcando exatamente onde clicar.

O que o sistema faz

  • Cadastra pessoas autorizadas a entrar em ambientes controlados;
  • Controla acesso por digital (biometria do T50M) e/ou senha (6 dígitos);
  • Grava cada tentativa com hora, ambiente, pessoa e método;
  • Liga câmeras IP ao acesso — cada entrada gera um trecho de vídeo associado;
  • Mantém auditoria completa de tudo que cada administrador faz.

Quem usa

Administradores cadastrados no banco. Não há cadastro de administrador pela tela — por segurança, só o setor de TI cria admins via banco direto (veja seção 14).

Antes de começar Você precisa de: navegador moderno (Chrome, Edge, Firefox), endereço do servidor do painel (algo tipo http://10.0.0.5:8080) e login/senha de admin entregues pela TI.

5.4 Vincular a um ambiente

Importante: a pessoa fica com status inativo até ser vinculada a pelo menos um ambiente. Veja seção 10 pra fazer isso.

Diferença dos dois modos de acesso
  • Digital + Senha: exige ambos. Pessoa digita ID + dedo + senha. Mais seguro, padrão militar.
  • Somente Senha: só ID + senha. Útil pra terceirizados, dedo machucado ou casos onde biometria não vale a pena cadastrar.

7. Trocar o modo de acesso

Às vezes você precisa mudar uma pessoa de Digital + Senha pra Somente Senha (dedo machucado) ou o contrário (pessoa que ficou fixa, vale cadastrar biometria).

  1. Entre no detalhe da pessoa

    Menu Pessoas → cartão da pessoa → "Ver Perfil".

  2. Clique em "🔄 Trocar Modo"

    Botão laranja no topo do detalhe. Abre confirmação mostrando pra qual modo vai mudar.

  3. O sistema cuida do resto

    Digital + Senha → Somente Senha: a biometria é desligada no T50M (volta a aceitar só senha). O template biométrico fica guardado no banco — se voltar pra Digital + Senha depois, é restaurado em segundos sem precisar cadastrar de novo.

    Somente Senha → Digital + Senha:

    • Se já tinha biometria antes: restauração automática a partir do template guardado.
    • Se nunca cadastrou: a pessoa precisa fazer um primeiro acesso no T50M com o dedo pra criar o template (3 leituras do dedo).
Capacidade do T50M Pra trocar pra Digital + Senha, cada T50M do ambiente precisa ter espaço (menos de 1000 digitais cadastradas). Se algum estiver cheio, o sistema bloqueia a troca e mostra qual T50M está lotado.

11. Adicionar uma câmera

Cada ambiente pode ter uma ou mais câmeras IP associadas. Quando alguém acessa, o sistema procura no servidor da câmera a gravação correspondente (via ONVIF) e guarda o link no histórico.

  1. Vá pra Câmeras no menu lateral

    (ou abra o detalhe de um ambiente e use a seção "Câmeras" dele)

  2. Clique em "+ Nova Câmera"

  3. Preencha o formulário

    CampoO que preencher
    NomeEx: "Câmera Frontal Sala Op"
    AmbienteEscolha o ambiente da câmera
    TipoInterna ou Externa
    URL RTSPEndereço pra visualização ao vivo. Formato: rtsp://usuario:senha@IP:554/.... A TI fornece esse endereço.
    URL HLS (opcional)Endereço HLS pra visualização web. Se não preencher, a aba "Ver ao Vivo" mostra placeholder.
    Endereço ONVIFEndpoint ONVIF (ex: http://192.168.0.50:80/onvif/device_service). É por aqui que o sistema busca gravações.

  4. Cadastre

    A partir daí, cada acesso liberado no ambiente da câmera gera URL de gravação que aparece no Histórico.

Câmera offline ou URL errada? Se a câmera não responder ao ONVIF, o campo "Gravação" no histórico fica vazio (—). A gravação não é perdida — ela está na câmera; o painel só não consegue mostrar a URL.

14. Administradores

A tela Administradores lista quem tem acesso ao painel. Dá pra ver quem é, editar dados auxiliares (nome, email, cargo) e ver os últimos 20 logs de cada admin.

Não dá pra criar admin pela tela Por decisão de segurança militar, criar/excluir admin e trocar senha só são feitos via acesso direto ao banco (com o GerarHash + DBeaver). Veja seção 8 do DeploymentGuide.html, ou peça pra TI.

O que dá pra fazer aqui

  • Listar todos os admins (login e nome)
  • Buscar por login ou nome
  • Ordenar por coluna
  • Clicar num admin → ver detalhes (CPF, email, cargo, telefone, data criação) e os últimos 20 logs dele
  • Editar dados auxiliares (CPF/email/cargo/telefone) — não muda login nem senha

16. Sair (Logout)

Use o botão Sair na parte de baixo do menu lateral (vermelho). Isso encerra a sessão e leva pra tela de login.

Sessão expira sozinha Mesmo sem você sair, o token de autenticação expira em 8 horas. Depois disso, qualquer ação leva pra tela de login de novo.

17. Perguntas Frequentes

Cadastrei a pessoa mas ela não consegue entrar no T50M

Provavelmente ela ainda não foi vinculada a um ambiente. Pessoa cadastrada mas sem ambiente fica inativa e nem aparece no T50M. Veja seção 10.

A pessoa aparece como ativa no painel mas o T50M nega o acesso

Pode ser que a sincronização com o T50M ainda não rodou. O SincronizadorT50 roda a cada 10 segundos. Espere 30s e tente de novo. Se persistir, verifique se o T50M está online (Dispositivos T50).

Esqueci a senha de uma pessoa, consigo ver?

Não. Senhas são guardadas cifradas. Você só consegue reenviar por email (botão "Reenviar Senha" no detalhe da pessoa, seção 6) ou copiar/anotar no momento do cadastro (seção 5).

O email do cadastro foi enviado mesmo? Como confirmar?

Logo após clicar em "Cadastrar", a modal mostra um banner: verde = enviado, amarelo = falhou, azul = enviando. Se ficar amarelo, use o botão "Copiar" pra pegar ID + senha e entregar manualmente, e avise a TI pra revisar o SMTP.

Por que algumas tentativas no histórico não têm vídeo (—)?

3 motivos possíveis: (1) o ambiente não tem câmera cadastrada; (2) a câmera está offline; (3) o ONVIF da câmera não respondeu no tempo configurado.

Quem pode mudar a chave AES, a chave JWT etc.?

Só o setor de TI, alterando variáveis de ambiente no servidor. Não tem tela pra isso (proposital).

Como faço backup do banco?

O sistema já faz backup do banco SQLite todo dia ao iniciar (cópia em banco_backup.db). Pra cópia adicional: copie o arquivo banco.db direto no servidor com o sistema parado.

Posso usar pelo celular?

Tecnicamente sim — o painel é responsivo. Mas como é uso interno e operacional, recomendado desktop pra cadastros e celular só pra consulta rápida.
